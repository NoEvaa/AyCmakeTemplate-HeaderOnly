# C++ Cmake Template: Header-only

A cmake template for C++ header-only library.

## 适用场景

- header-only库

- 模板库/宏定义库

更多相关内容请移步[AyCmakeTemplate](https://gitee.com/ayin-tech/ay-cmake-template)。

## 快速开始

### 构建

```bash
sh build.sh
sh build.sh Release
```
